Descrição

Este projeto consiste em uma aplicação mobile desenvolvida com React Native, com o objetivo de permitir o cadastro e a visualização de tarefas.

O aplicativo possui um fluxo simples e funcional, onde o usuário pode adicionar novas tarefas e visualizá-las em uma lista.



 Funcionalidades

  Listagem de tarefas
  Cadastro de novas tarefas
  Navegação entre telas
  Validação de formulário (campo obrigatório)
  Uso de componentes reutilizáveis



 Telas do Aplicativo

Tela Inicial: Exibe a lista de tarefas cadastradas
Tela de Cadastro: Permite adicionar uma nova tarefa



Tecnologias Utilizadas

 React Native
 JavaScript
 React Navigation

Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npx expo start
```

---

 Observações

 É necessário ter o Node.js e o Expo CLI instalados.
 O aplicativo pode ser executado em emulador ou no celular usando o app Expo Go.
